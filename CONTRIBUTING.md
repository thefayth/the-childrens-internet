# Contributing

This is not an open contribution repository.

The public surface is provided for review, orientation, project legitimacy, and controlled sharing. Source code, implementation workflows, operational systems, and private materials are not open for public contribution.

For collaboration inquiries, contact Faith Cheltenham through FaithCheltenham.com.

Do not submit pull requests containing copied source code, scraped materials, credentials, screenshots of private systems, child-related personal data, or unpublished documents.
