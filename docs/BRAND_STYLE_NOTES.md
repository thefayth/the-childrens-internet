# Brand Style Notes

## Brand Direction

The Children's Internet should feel protective, imaginative, human, and accountable. It should not feel like generic SaaS, surveillance software, or a fear-based safety product.

## Visual Qualities

- Storybook warmth
- Clear child-first optimism
- Family trust
- Calm safety operations
- Bright but not chaotic color
- Playful forms with serious boundaries

## Color Direction

- Warm sun: `#ffd86f`
- Soft sky: `#addfff`
- Mint: `#b9efcb`
- Coral: `#ff9ea2`
- Deep ink: `#24415d`

## Typography Direction

Rounded, friendly display type paired with readable body text. The current microsite uses a playful rounded visual language that the export preserves through generated assets.

## Voice

Confident, protective, and emotionally clear. Avoid overclaiming live product capabilities.
