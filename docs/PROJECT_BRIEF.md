# Project Brief

## What This Is

The Children's Internet is a child-first internet concept presented as a public, storybook-style microsite and project surface. It imagines a safer digital layer where children can explore, learn, join clubs, create, and connect inside boundaries designed for childhood.

## Why It Matters

Open-web defaults are not designed around childhood. This project frames an alternative: protected exploration, family guidance, age and trust gates, AI screening, human moderation, and accountable escalation working together as one system.

## Current Public Expression

The current working surface is a static presentation microsite at:

https://faithcheltenham.com/kidsinternet/

It contains:

- A vision/home route
- An interactive narrative demo
- A safety-system route
- A slide-style pitch mode
- Original child-safe storybook artwork

## Current Product Boundary

This is not a live child account system. The backend product, if pursued, would require a separate plan for identity, consent, moderation, parent dashboards, data retention, compliance, institutional deployment, and operations.
