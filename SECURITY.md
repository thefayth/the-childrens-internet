# Security Policy

This repository is a public documentation and project-surface package. It intentionally excludes source code, secrets, credentials, deployment adapters, private workflows, and operational infrastructure.

## Reporting

If you believe public material in this repository exposes sensitive information or creates a safety risk, contact Faith Cheltenham through FaithCheltenham.com.

Do not open public issues containing credentials, private personal information, protected child-safety details, or exploit instructions.

## Scope

In scope:

- Accidental exposure of sensitive project details.
- Broken public/private boundary language.
- Public asset or documentation issues that could mislead users about current product status.

Out of scope:

- Requests for source code access.
- Requests for credentials or operational details.
- Attempts to bypass private systems.
- Claims about backend features that are not publicly implemented.
